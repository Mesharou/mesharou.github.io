Impact of seamounts

  • Edit the file ana_grid.F and add a seamount (or a chain of seamounts) in the middle of your domain.

  • Edit the file param.h and increase the number of points in the vertical:

      #if defined BASIN
  parameter (LLm0=120,   MMm0=100,   N=20)

  • Plot a vertical section of temperature or density across the domain

  • Plot the relative vorticity at the surface.

  • Plot a horizontal section of relative vorticity at the seamount mid-depth.

Impact of explicit/implicit viscosity

We will re-run the turbulent case (with or without without topography) with different numerical choices and compare the evolution of the gyre and the barotropic vorticity budget. You can find more info on the available advective schemes here

  • Start from the turbulent BASIN case at the end of Activity 2 (with vorticity diagnostics included)

  • Run the following setups:

    • Simulation 1: choose numerical options to have no implicit horizontal viscosity/diffusion and an explicit horizontal viscosity/diffusion of 10000 m2/s

    To have no implicit horizontal viscosity/diffusion, you need to pick a centered advective scheme (C2, C4, or C6). For example:

    # define  UV_HADV_C4
# define  TS_HADV_C4

    To use explicit horizontal viscosity/diffusion, you need to define:

    # define  UV_VIS2
# define  TS_DIF2

    And choose viscosity/diffusion values in the croco.in.

    • Simulation 2: choose numerical options to have only implicit horizontal viscosity/diffusion and no explicit horizontal viscosity/diffusion

    • Simulation 3: choose numerical options to have no horizontal implicit nor explicit viscosity/diffusion

  • For each simulation, compare the velocity (or vorticity) field. Do you see any differences?

  • Plot the different terms of the barotropic vorticity budget

  • Eventually plot the different terms of the kinetic energy budget averaged over the last year of the simulation.
